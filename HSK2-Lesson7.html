HSK2 Lesson 7 你家离公司远吗？

🎯 Learning Objectives
  • Ask and answer questions about distance
  • Use key sentence structures
  • Respond in simple conversations
🧠 Key Vocabulary
  • 离 (lí) — from
  • 远 (yuǎn) — far
  • 近 (jìn) — near
  • 家 (jiā) — home
  • 公司 (gōngsī) — workplace
🗣️ Sentence Pattern

A: 你家离公司远吗？

B: 不远 / 很远 / 挺近的

🎲 Pair Practice

Ask and answer:

你家离学校远吗？

很远 / 不远 / 很近

Extension: 我坐车 / 坐地铁去公司
